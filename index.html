CC5. Accessing data: Scraper and API

API Challenge: World Bank GDP Data

API Description

Base URL:

https://api.worldbank.org/v2/country/

API Structure:

  • Countries: VNM;THA;IDN;PHL;MYS (Vietnam, Thailand, Indonesia, Philippines, Malaysia)
  • Indicator: NY.GDP.MKTP.KD.ZG (GDP growth annual %)
  • Format: format=json (Returns data in JSON format)
  • Results per page: per_page=1000 (Maximum results returned)
  • Date range: date=1990:2025 (Data from 1990 to 2025)

Complete URL used in chart:

https://api.worldbank.org/v2/country/VNM;THA;IDN;PHL;MYS/indicator/NY.GDP.MKTP.KD.ZG?format=json&per_page=1000&date=1990:2025

How it works: The World Bank API retrieves economic indicators for specified countries. The URL structure follows the pattern: base_url/[countries]/indicator/[indicator_code]?[parameters]. The API returns JSON data with two arrays: metadata (index 0) and actual data (index 1), which is accessed in Vega-Lite using "format": {"property": "[1]"}.
